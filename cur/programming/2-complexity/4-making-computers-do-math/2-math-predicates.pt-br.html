Uma biblioteca de software é uma coleção de procedimentos que podem ser usados em programas.

O uso de bibliotecas simplifica o desenvolvimento de novos programas. Quando você utiliza procedimentos que já sabe que funcionam corretamente, reduz a quantidade de tempo necessária para a codificação, a quantidade de possíveis erros que seu código pode ter e a quantidade de testes que você precisa realizar em seu projeto.

  1. Exporte os blocos matemáticos do seu projeto U2L3-Predicados.

    Exportar Blocos

    1. Abra o projeto Snap! que contém os blocos que deseja exportar. (Neste caso, seu projeto U2L3-Predicados).
    2. Escolha "Exportar Blocos..." no menu Arquivo do Snap! ().
    3. Selecione apenas o(s) bloco(s) que deseja exportar. (Neste caso, você precisa de , , , e .)
      Você pode desmarcar os blocos que não precisa exportar, ou clicar com o botão direito no fundo, escolher "nenhum" e depois selecionar apenas os que deseja.
      4. As bibliotecas e os projetos do Snap! são exportados como arquivos XML.
    4. Clique em "Aceitar". Um arquivo XML será baixado.
  2. Importe esses blocos para o seu projeto U2L4-BibliotecaMatemática, teste-os, resolva quaisquer problemas e salve-o.

    Importar Blocos

    1. Abra o projeto Snap! que contém os blocos que deseja exportar. (Neste caso, seu projeto U2L4-BibliotecaMatemática).
    2. Importe o arquivo XML arrastando o arquivo XML da sua pasta de downloads para a janela do Snap!.
      Você também pode importar o arquivo XML selecionando "Importar..." no menu Arquivo do Snap!, localizando o arquivo XML no seu computador e clicando em "Abrir".
    3. Faça testes nos blocos que você importou:
      1. Encontre os blocos importados no final das paletas que os contêm.
      2. Clique em cada bloco importado para verificar se eles são executados corretamente.
Você está criando uma biblioteca matemática. O Snap! possui bibliotecas para muitas coisas, por exemplo, uma que dá acesso a diferentes partes de palavras e frases...

Uma interface de programação de aplicativo (API) documenta o que um programador precisa saber sobre o uso da biblioteca: é uma descrição do propósito, das entradas e saídas de cada procedimento (mas não de seus algoritmos).

O que é uma API web?

Um tipo comum de API é uma API web em que a biblioteca existe no computador de outra pessoa. Por exemplo, a API do Google Maps descreve como incorporar um mapa do Google em seu próprio site.

Uma API web é apenas uma chamada a um procedimento em outra máquina. Por exemplo, estas são três notações diferentes para a mesma chamada de procedimento, que busca o número de buscas por "BJC" nos EUA:

  • https://trends.google.com/trends/explore?q=BJC&geo=US
  • EXPLORE (q = BJC, geo = US)
