Mantendo Itens de uma Lista

Nesta página, você usará predicados com o bloco keep para manter os itens em uma lista que possuem características específicas.

Como você sabe, predicados podem ser usados com condicionais para decidir quando fazer algo; eles também podem ser usados com o bloco keep para decidir quais coisas manter. O bloco recebe uma lista e um predicado como entrada e relata uma nova lista mantendo apenas os itens da lista de entrada que atendem à condição descrita pelo predicado.

Por exemplo, a seguinte expressão encontrará palavras da lista de palavras cuja primeira letra é v. O espaço em branco na entrada é onde cada item da lista vai para decidir se ele será mantido.

String e Índice
    AAP-1.C.4
  • Uma string é uma sequência ordenada de caracteres.
    • AAP-1.C.3 (strings)
  • Você já aprendeu a palavra índice para a posição de um elemento em uma lista. A mesma palavra é usada para a posição de um caractere em uma string (por exemplo, uma letra em uma palavra). Portanto, a primeira entrada para o bloco letra é o índice.

Você escreve o predicado que faz a verificação e o keep aplica o predicado a cada item na lista de entrada e, em seguida, relata a lista de itens que tornam seu predicado verdadeiro.

Observe que o espaço em branco na entrada do predicado é obrigatório; é aqui que cada item da lista vai à medida que é verificado pelo predicado.

O bloco letra na paleta parece com . Você precisa excluir o "mundo" para que haja um espaço em branco onde você precisa.

  2. Experimente esses exemplos de keep. Discuta e, em seguida, explique por escrito o que essas expressões estão fazendo.

O primeiro exemplo mantém entradas de um certo comprimento; cada palavra na lista é perguntada se seu comprimento é 5 e apenas aquelas com cinco letras são relatadas. O segundo exemplo mantém entradas que não são números; cada item nesta lista é perguntado "Este item é não um número?" e apenas os elementos que não são números são relatados.

Existem dois blocos comprimento no Snap!. Um () é projetado para strings, e o outro () é projetado para listas.
