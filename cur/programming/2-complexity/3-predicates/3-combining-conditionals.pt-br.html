AAP-2.I.1

Uma declaração condicional aninhada é uma declaração se ou se senão dentro de outra declaração se senão.

Olhe novamente para os dois primeiros exemplos no problema anterior. Como a expressão predicativa a = b relatará verdadeiro quando eles forem iguais e falso caso contrário, não é necessário usar a declaração condicional aninhada no primeiro exemplo, e o segundo exemplo usando o predicado dentro da parte senão é suficiente. No entanto, às vezes (especialmente quando você não está construindo um predicado), pode ser útil usar declarações condicionais aninhadas.
