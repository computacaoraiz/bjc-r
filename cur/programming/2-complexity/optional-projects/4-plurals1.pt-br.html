Siri e outros robôs são programados para gerar linguagem. Siri fala; alguns robôs escrevem. Você desenvolveu um programa, fofoca, que escreve. Neste projeto, você estenderá as habilidades do computador com a linguagem, desenvolvendo um bloco de repórter, plural, que recebe um substantivo como entrada e soletra corretamente o plural.

Soletrar um plural pode parecer quase automático para uma pessoa, mas não é simples: um computador precisa de um algoritmo para realizar essa tarefa.
Este projeto usa blocos já carregados para você na biblioteca "Palavras, frases" do Snap!. Você pode carregar uma biblioteca no menu Arquivo do Snap! em "Bibliotecas...".
