Siri e outros robôs são programados para gerar linguagem. Siri fala; alguns robôs escrevem. Você desenvolveu um programa, fofoca, que escreve. Neste projeto, você ampliará as habilidades do computador com a linguagem desenvolvendo um bloco repórter, plural, que recebe um substantivo como entrada e soletra e gera corretamente o plural

Soletrar o plural pode parecer quase automático para uma pessoa, mas não é simples: um computador precisa de um algoritmo para realizar essa tarefa.
Este projeto usa blocos já carregados para você na biblioteca "Palavras, frases" do Snap!. Você pode carregar uma biblioteca no menu Arquivo do Snap! em "Bibliotecas...".

    Experimente todas as suas expressões (os nove scripts de uma linha) e experimente suas entradas até ter certeza de que entendeu o que elas fazem. Para cada expressão, diga o que o script faz em geral e não apenas o que faz com essa entrada específica. (Exemplo: reporta "y", mas você deseja dizer qual última letra de reportará se, digamos, "2017" for sua entrada.)
    2. Você não precisará do estilo para o projeto, mas você pode achar interessante tentar não apenas com "quando" como entrada, mas com "onde", "o quê", "por que". Os linguistas usam evidências como esta para compreender a história de uma língua.
