Making a Kaleidoscope

Neste projeto, você criará um aplicativo que permite desenhar como um caleidoscópio.

Um sprite está seguindo o mouse. Três outros sprites refletem o movimento desse sprite de maneiras ligeiramente diferentes.

  1. "Caleidoscópio U2"
  2. Primeiro, crie e teste o código para fazer o primeiro sprite seguir o mouse e desenhar uma linha traçando seu caminho.
    1. Use um evento para acionar o comportamento do caleidoscópio para que os usuários não precisem abrir cada sprite e clicar em cada script para iniciar o desenho.
    2. Faça o sprite desenhar apenas quando o mouse estiver pressionado (você precisará desmarcar a caixa "arrastável" para que o Snap! não pense que você está tentando arrastar o sprite ao clicar. )

    Precisa de um lembrete?

    Você aprendeu como fazer o sprite seguir o mouse em Unidade 1 Laboratório 5 Página 1: Sprite seguindo o mouse. Você aprendeu como detectar um evento em Unidade 1 Laboratório 1 Página 2: Programando um jogo. E você aprendeu como desenhar apenas quando o mouse está pressionado usando if else com mouse down? em Unidade 2 Laboratório 3 Página 1: O que é um predicado?.

  3. Crie um segundo sprite que espelhe o movimento do primeiro.

    Uma maneira de criar o segundo sprite é duplicar o primeiro e modificá-lo. Clique com o botão direito no botão do sprite no curral de sprites (no canto inferior direito da janela Snap!) e selecione "duplicar" no menu de contexto que aparece. O novo sprite terá os mesmos scripts do original, por isso a duplicação é mais fácil do que criar novos sprites do zero.

    Precisa de uma dica sobre como fazer o segundo sprite espelhar o primeiro?

    Você vai querer que o segundo sprite ou vá para a mesma posição x e para a posição y inversa ou para ir para a posição x inversa e para a mesma posição y.

    Precisa de outra dica?

    Use dentro de .

  4. Teste seu código e depure quaisquer problemas.
    Você pode querer usar um evento (como pressionar a tecla "c") para dizer ao programa para limpar o palco e parar todos os scripts. Isso facilitará o teste e a depuração.
  5. Crie mais dois sprites que espelhem os dois que você tem.

    Se o seu segundo sprite espelhar o primeiro horizontalmente, o terceiro e o quarto sprites deverão espelhar esses sprites verticalmente e vice-versa.

  1. Experimente algumas das ideias a seguir ou faça suas próprias alterações no caleidoscópio.
    • Troque as canetas:
        Você aprendeu sobre definir a cor da caneta e definir o tamanho da caneta em Unidade 1 Laboratório 3 Página 1: Explorando o movimento.
      • Adicione algumas cores.
      • Ajuste os tamanhos das canetas.
    • Crie mais sprites:
      • Adicione novos sprites que usam mouse x como entrada y e mouse y como entrada x.
      • Adicione novos sprites que estejam afastados de outro sprite.
    • Crie suas próprias variações.
