Mantendo a Pontuação com Variáveis Globais

Nesta página, você usará uma variável global para manter a pontuação no seu jogo Click Alonzo.

No seu Jogo de Adivinhação de Números, você usou variáveis de script para armazenar informações (o valor do número secreto) que eram necessárias apenas por um script. Para manter a pontuação no Click Alonzo, mais de um script precisará acessar essa informação, então você usará uma variável global.

: Variável Global

Uma variável global é uma variável utilizável por todos os scripts no programa.

Quando as variáveis globais são necessárias?

Na maioria das vezes, variáveis de script são uma escolha melhor, porque não podem ser alteradas por outra parte do programa, reduzindo assim a chance de bugs. Mas se vários scripts precisarem acessar uma variável ou se as informações naquela variável precisarem ser salvas com o seu projeto, então use uma variável global.

  1. Abra o seu projeto U1L1-ClickAlonzo, jogue o jogo e revise o código para lembrar como ele funciona.
  2. "U2L1-ClickAlonzo" Certifique-se de renomear o seu projeto com "U2" no nome do arquivo.
    3. Se você salvar um novo trabalho com um nome antigo de arquivo, o Snap! irá substituir o seu projeto original. Use a opção "Salvar como..." no menu Arquivo do Snap! para salvar com o novo nome. Quando você carregar um projeto antigo e estiver prestes a fazer uma grande alteração, adquira o hábito de salvar primeiro, com o novo nome, antes de fazer qualquer alteração.
  3. Crie uma variável global chamada pontuação. Veja como fazer isso.

    4. Criando uma Variável Global

    1. Clique em na paleta de Variáveis. (Não é um bloco; você não pode arrastá-lo para a área de programação.)
    2. Digite o título da sua variável. Neste caso, é pontuação.
    3. Clique em "OK".
  4. Use a variável pontuação para acompanhar a pontuação do jogador:
    1. Inicialize pontuação como 0 no início do jogo.

      2. Definir o valor inicial de uma variável é conhecido como inicializar  a variável.

      AAP-1.B.1
    2. Faça o programa alterar a pontuação em 1 sempre que o sprite for clicado.
      3. (o que significa ) seria escrito como 
      pontuação ← pontuação + 1
      ou .
  5. Teste e depure. Jogue o jogo o suficiente para garantir que a variável de pontuação funcione.
  6. Neste momento, o jogo não tem uma maneira de vencer, mas você vai mudar isso. Decida a pontuação que você usará para testar se o jogador venceu.
  7. Use uma condicional para determinar quando o jogador atinge essa pontuação. Quando eles atingirem:
    1. Faça Alonzo parar de pular.
    2. Faça Alonzo aparecer novamente. (Você terá que redefinir o efeito fantasma.)
    3. Parabenize o jogador por alguns segundos.
  8. Teste e depure. Alternem-se jogando e corrigindo quaisquer bugs.

    Dica de Depuração: Exibição de Variáveis

    Ao depurar o seu código, pode ser útil ver o valor de uma variável em diferentes pontos do programa. Isso é feito de forma diferente para variáveis locais e globais:

    • As variáveis globais têm caixas de seleção que controlam a exibição do observador da variável no palco (mostrado abaixo).
    • Se for uma variável local (de script), você pode usar o bloco exibir variável dentro do script para exibir um observador no palco (e usar ocultar variável para escondê-lo novamente).

  1. Se o sprite se mover sem ser clicado, faça a pontuação diminuir 1.
  2. Jogo muito difícil? Aumente a pontuação em 2 por acerto. Mudança muito grande? Aumente em 3 por acerto, diminua em 2 por erro. Você pode fazer ajustes arbitrariamente finos. Esse processo é chamado de ajuste de jogo.
