Escolhendo uma Fantasia

No Click Alonzo, o jogo alternava entre duas versões diferentes da fantasia do Alonzo para informar ao jogador se eles clicaram com sucesso no Alonzo. Nesta página, você permitirá ao usuário do seu programa Click Alonzo escolher a fantasia do sprite.

    Você aprendeu como adicionar fantasias em Unidade 1 Laboratório 2 Página 3: Personalização e Depuração.
  1. Se ainda não estiver aberto, abra o seu projeto U2L1-ClickAlonzo.
  2. Configure o que você precisa para esta nova versão:
    1. Adicione várias fantasias.
    2. Exclua a fantasia de Alonzo de costas.
    3. Remova o código no programa que altera a fantasia quando você clica no sprite.
  3. Experimente o bloco usando entradas diferentes para alternar manualmente a fantasia do sprite algumas vezes.
Se você estiver jogando o seu jogo e quiser mudar a fantasia, pode simplesmente escolher a fantasia que deseja pelo nome, no menu suspenso do bloco mudar para a fantasia. No entanto, imagine que você está dando este jogo para um amigo não programador jogar. Você quer programar a seleção de uma nova fantasia dentro do jogo, para que seu amigo não precise saber nada sobre blocos.

Quando você usou item de antes (por exemplo, dentro de quem e faz o quê), você estava acessando itens da lista de forma aleatória. Mas você também pode selecionar um item específico especificando sua posição.

: Índice

O número da posição é chamado de índice do item na lista.

Nesta lista, 1 é o índice do item "maçã", 2 é o índice do item "melão" e assim por diante.

AAP-1.D.8

No Snap! e no exame AP, o índice é sempre um número inteiro (1, 2, 3, 4, etc.). É um erro usar um índice menor que 1 ou maior que o comprimento da lista.

Listas podem conter qualquer coisa: letras ou palavras, fantasias, outras listas ou até mesmo blocos. Você pode usar meu para relatar uma lista de fantasias para o seu sprite.

O bloco meu parece até você selecionar "fantasias" no menu suspenso dele.

E você pode usar item de junto com meu para selecionar uma fantasia em uma posição específica na lista.

Alguns blocos (como item de) têm espaços de entrada que esperam uma lista. Você pode perceber isso porque o espaço de entrada se parece com uma lista:
