Adicionando itens a uma lista

AAP-1.C.2

Um elemento é outro nome para um item em uma lista. (Se o mesmo valor estiver na lista duas vezes, isso conta como dois elementos diferentes.) Cada elemento possui um índice (posição) exclusivo na lista.

Você pode usar o ou bloco para adicionar elementos a uma lista.
  • Inserir coloca o novo item antes do local especificado.
  • Adicionar coloca o item depois do último item existente.

AAP-1.D.6

Atribuir uma lista a uma variável permite usar um nome para representar todos os elementos de uma lista como uma unidade.
