Laboratório 3: Arte Moderna com Polígonos

1.3.4
AAP-3.C.1, AAP-2.K.2
A definição do procedimento para o comando catavento personalizado

seria escrito como
        Cata-vento de PROCEDIMENTO(numberOfBranches)
        {
             REPETIR numberOfBranches TIMES
             {
                 mover(100)
                 mover(-37)
                 turn_clockwise(360 / númeroDeRamos)
             }
        }
ou
Os procedimentos 
move()
e 
turn_clockwise()
não estão integrados na linguagem do AP portanto, eles são escritos em letras minúsculas, como outros procedimentos definidos pelo programador.
Observe que o bloco de chapéu, , seria escrito como 
PROCEDURE pinwheel(numberOfBranches)
. A palavra 
PROCEDURE
indica que aquela linha do código é como um bloco de chapéu; o nome da variável entre parênteses nessa linha é a entrada que o procedimento recebe.
1.3.4
AAP-3.A.5
Esta instrução seria escrito como 
Pinwheel(6, 80, 20)
ou .

Você pode ouvir as pessoas usarem o termo "pseudocódigo" para se referir a essa pseudolinguagem usada no exame AP CS Principles, mas não é pseudocódigo. Pseudocódigo não é uma linguagem de programação, é o uso da linguagem humana normal para descrever um algoritmo.
