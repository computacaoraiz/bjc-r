Compartilhar seu jogo

O jogo ainda não está quase terminado, mas está terminado o suficiente. Nesta página você irá compartilhar seu projeto com o seu parceiro e com amigos que possuam smartphones.
  1. Crie um link para compartilhar seu app.

    Criando um link

    1. Tenha certeza de que seu projeto está salvo.
    2. Clique em "Abrir..." ("Open...") no menu Arquivo (File) do Snap!, .
    3. Selecione o projeto que você deseja compartilhar na lista de projetos (provavelmente o nome do seu projeto será "U1L1-ClickAlonzo").
      4. Para localizar um arquivo você pode digitar na barra de pesquisa no topo para filtrar os arquivos e ver apenas a lista com nomes que batem com o que você digitou (veja o exemplo à direita, onde apenas arquivos com "U1" são exibidos).
    4. Clique em "Compartilhar" ("Share") e depois clique em "Sim" ("Yes") para efetivar o compartilhamento. Depois clique em "Cancelar" ("Cancel") para fechar o menu.
    5. Repare agora que a URL (o link) na janela de seu navegador foi alterado. Copie esse nova URL e envie por e-mail para você mesmo. Quando você tiver tempo poderá enviar esse link para outras pessoas compartilhando assim o seu projeto.
      6. Clique para ver imagens que mostram onde está a URL e como ela é alterada.

      se você alterar alguma coisa no seu projeto após compartilhar a URL, as mudanças serão exibidas automaticamente no projeto que as outras pessoas acessam. Você não precisa compartilhar novamente o projeto depois de cada modificação.

  2. Se você está trabalhando com um parceiro, tenha certeza de que vocês dois tenham uma cópia do projeto.

    Salvando uma cópia do projeto

    1. Troque quem está logado no Snap!
      1. Abra o menu nuvem, , e escolha "Sair" ("Logout"). Não feche a janela do Snap! ou saia do navegador.
      2. Deixe seu parceiro logar no Snap!.
    2. Deixe seu parceiro salvar o projeto na conta dele pois, assim, vocês dois terão uma cópia do projeto.
      3. Atenção: seu parceiro terá uma cópia — não o arquivo compartilhado. Assim, se um de vocês alterar o projeto na sua própria conta, essas alterações não serão aplicadas à copia do projeto na conta do outro; Assim, se um de vocês alterar o projeto, o outro precisará salvar uma cópia novamente para obter a versão mais nova com as alterações.

Colocando o jogo em seu telefone

Os quadro de cor azul "Se houver tempo" têm atividades opcionais que não são necessárias para este curso. Se você terminar tudo na página mais cedo, realize as atividades indicadas antes de avançar para a próxima página do laboratório. Se você tem um smartphone, com certeza vai querer realizar essa atividade.
  1. Carregue o seu jogo em um telefone celular.

    Carregando seu projeto Snap! no Apple iOS

    1. Em seu telefone celular abra o Safari e digite a URL (link) para o seu projeto.
    2. Na parte de baixo da tela do Safari, toque no ícone de compartilhamento, , e selecione a opção "Adicionar na Tela Inicial" no menu que será exibido.

    Carregando seu projeto Snap! no Android

    1. Em seu telefone celular abra o Chrome e digite a URL (link) para o seu projeto.
    2. Toque no ícone de configurações, ☰, e então escolha a opção "Adicionar à tela inicial" no menu que será exibido.
  2. Certifique-se de que você pode abrir o projeto como qualquer outro aplicativo na tela inicial do seu telefone.
