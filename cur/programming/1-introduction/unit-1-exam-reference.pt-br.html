Laboratório 2: Fofoca

1.2.2
AAP-3.A.6
Não há algo exatamente igual a ou no Exame do AP, pois eles não possuem sprites e balões de fala, mas a maneira como eles mostram esse texto ao usuário é 
EXIBIR(fofoca())
se for escrito como texto ou se for mostrado como blocos.

Você não precisará ser capaz de escrever código nessa notação no Exame do AP. Você só precisa ser capaz de ler para poder responder perguntas sobre isso.

1.2.4
    AAP-3.A.7, AAP-3.C.2

Laboratório 3: Arte Moderna com Polígonos

1.3.4
AAP-3.C.1, AAP-2.K.2
A definição do procedimento personalizado pinwheel

seria escrita como
PROCEDURE pinwheel(numeroDeRamos){    REPETIR numeroDeRamos VEZES    {        mover(100)        mover(-37)        girar_sentido_horario(360 / numeroDeRamos)    }}
ou
Os procedimentos 
mover()
e 
girar_sentido_horario()
não são incorporados à linguagem do AP, portanto, são escritos em letras minúsculas como outros procedimentos definidos pelo programador.
Observe que o bloco de encaixe, , seria escrito como 
PROCEDURE pinwheel(numeroDeRamos)
. A palavra 
PROCEDURE
indica que essa linha de código é como um bloco de encaixe; o nome da variável entre parênteses nessa linha é a entrada que o procedimento aceita.
1.3.4
AAP-3.A.5
Esta instrução seria escrita como 
pinwheel(6, 80, 20)
ou .

Você pode ouvir as pessoas usarem o termo "pseudocódigo" para se referir a essa pseudo-linguagem usada no Exame de Princípios de Ciência da Computação do AP, mas não é pseudocódigo. Pseudocódigo não é uma linguagem de programação, é o uso da linguagem humana normal para descrever um algoritmo.

Laboratório 5: Siga o Líder

1.5.2
AAP-2.K.3
A linguagem usada no Exame do AP não permite espaços nos nomes de entradas (como number of fish) ou em procedimentos definidos pelo programador (como mouse y, que não está incorporado à linguagem deles). Portanto, este exemplo os traduz para 
numFish
e 
MouseY()
.
A razão para o 
()
ou o retângulo após 
MouseY
é que 
MouseY()
é uma chamada de procedimento, mesmo que não aceite nenhuma entrada.

O script

seria escrito como

REPETIR ATÉ(mouseY() < 0){    EXIBIR(numFish)}
ou

Lembre-se, você não precisa aprender a escrever a linguagem inventada usada no Exame do AP. Você só precisa ser capaz de ler e responder perguntas sobre ela.
