Unidade 1 Laboratório 5: Siga o Líder, Página 1

Em muitos programas, os elementos visuais na tela precisam interagir — seguem uns aos outros, evitam uns aos outros, reagem de uma certa maneira quando se encontram, etc. Neste laboratório, você programará dois sprites. Um sprite seguirá seu mouse. O outro sprite perseguirá o primeiro. Quando eles se encontrarem, eles terão uma breve conversa.

Nesta página, você irá programar o primeiro sprite para seguir seu mouse e programar o segundo sprite para apontar na direção do primeiro sprite em preparação para segui-lo.

Programando um Segundo Sprite

  1. Crie um segundo sprite clicando no botão botão logo abaixo do palco.  
    O novo sprite, automaticamente nomeado Sprite(2), aparece no palco. É novo, então não tem scripts. As imagens dos sprites abaixo do palco permitem que você selecione o sprite que deseja programar.
  2. Para tornar seus nomes de sprites significativos, altere o nome de Sprite para Líder e altere Sprite(2) para Seguidor.

    Mudando o nome de um Sprite

    Procure o nome em uma caixa cinza acima da área de script. Clique dentro do nome e você poderá editá-lo.

  3. Enquanto um script está em execução, sua borda acende. Se o tempo de execução for curto, você pode não notar o breve flash.
    Certifique-se de ter o sprite Seguidor selecionado. Escreva um script para Seguidor para que, uma vez que a bandeira verde seja clicada, ela aponte para Líder para sempre.
  4. Ambos os sprites têm scripts que usam , então ambos desses scripts são executados quando é clicado.
    Click e verifique se seus scripts fazem o que você pretendia:
    • Líder sempre segue o mouse quando você move o mouse.
    • Seguidor permanece em um lugar, mas sempre aponta para Líder.
