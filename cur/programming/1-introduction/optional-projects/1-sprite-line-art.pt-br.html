Brinquedo Artístico

Neste projeto, você vai simular o famoso brinquedo infantil Grave um esboço®.

  1. Aqui estão algumas coisas para fazer no início do seu programa quando a bandeira verde for clicada:
    • Limpe todos os desenhos anteriores
    • Coloque o sprite de desenho no centro da tela
    • Aponte-o para cima
    • Defina a caneta de desenho para uma cor e tamanho de sua escolha
    • Abaixe a caneta para se preparar para desenhar
  2. Escreva o código que limpará os desenhos quando a tecla de espaço for pressionada pelo usuário.
  3. O usuário usará as teclas de seta para desenhar. Você precisará de um script para cada tecla, começando com um bloco quando tecla pressionada:

    Para cada uma das quatro teclas de seta, faça o sprite apontar na direção da seta e avance 10 etapas.
Aqui está como Snap! atribui ângulos para direções:
  • 0° é acima
  • 90° está certo
  • 180° está em baixo
  • -90° (ou também 270°) é deixado
  1. Altere seu código para desenhar apenas se o sprite estiver dentro das dimensões da tela: -240 a 240 na direção x (horizontal) e -180 a 180 na direção y (vertical).
  2. Dê ao seu usuário algumas opções de cores. Por exemplo, quando a tecla "1" é pressionada, a cor da caneta é redefinida para preto; quando a tecla "2" é pressionada, a cor da caneta é definida como vermelha; etc. (Sugestão: Carregar a biblioteca de giz de cera ("Forneça 100 cores selecionadas.")) Considere também usar algumas teclas para colocar a caneta para cima ou caneta para baixo.
  3. Dê ao seu usuário a opção de desenhar linhas diagonais. Por exemplo, escreva o código para mover o sprite em 45°, 135°, 225°, 315°, usando as teclas "w", "s", "a" e "q".
