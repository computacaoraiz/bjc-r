Desenho de Margarida

Neste projeto, você vai escrever código para desenhar o desenho geométrico mostrado abaixo, explorar suas variações e usá-lo para desenhar desenhos mais complexos.
  1. Discuta como esse projeto foi criado. Quais são os blocos de construção fundamentais? Como eles são organizados?

    2. Você pode assistir esta animação para ter algumas ideias.

  2. Todo o desenho é feito de círculos. Portanto, a primeira coisa a fazer é escrever o código que criará um círculo. Para fazer isso, crie um bloco personalizado Desenhar Círculo através da opção “Criar um bloco” do menu “Variáveis”. Dica: um polígono regular com 30 ou mais lados é uma boa aproximação para um círculo.
  3. Agora que você pode desenhar um círculo, você pode gerar o Daisy Design girando seu sprite um pouco no final de cada círculo desenhado. No desenho acima existem 24 círculos. Quanto o sprite deve girar cada vez que um círculo é desenhado para fazer um ciclo completo de 360° quando todos os círculos são desenhados? Crie um bloco personalizado Draw Daisy para fazer isso.
  1. Estenda seu código para que você possa criar os seguintes padrões.
  2. Considere as seguintes variações na cor e espessura da caneta para o Daisy Design. Observe que no primeiro desenho, a cor muda dentro de cada círculo, enquanto no segundo desenho, a espessura da caneta muda dentro de cada círculo. Estude o código e explore suas próprias variações.

  3. Crie suas próprias variações do Daisy Design e compartilhe com seus colegas.
