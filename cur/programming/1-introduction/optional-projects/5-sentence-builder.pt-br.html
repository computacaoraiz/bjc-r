Linguagem de modelagem

Neste projeto, você criará um programa capaz de criar uma linguagem mais complicada do que o projeto Gossip.

    • substantivos (coisas)
    • verbos (ações)
    • adjetivos (descritores de coisas)
    • advérbios (descritores de ação)
    • determinantes (identificadores—dizem qual coisa)
    • preposições (indicadores de relacionamento)

  1. Ele contém seis repórteres, cada um relatando uma palavra aleatória de uma lista de palavras nessa categoria.

  2. Experimente cada bloco para ver o que ele faz. Em seguida, altere essas listas como quiser para incluir seus próprios substantivos, verbos, etc.
  3. Crie um repórter chamado frase simples que relata uma frase simples usando o determinador aleatório, substantivo aleatório e verbo aleatório blocos.
  4. Frases mais complicadas podem ser construídas a partir de várias frases. Crie estes repórteres de frase:
    1. deve juntar palavras de duas categorias para fazer frases como "o sanduíche" ou "uma girafa", ou de três categorias para criar frases como "o garotinho", "uma girafa verde" ou "nosso velho sanduíche".
    2. deve combinar um preposição com uma frase nominal para criar frases como "near the little boy" ou "over a wise pizza."
      3. Usar frase nominal para definir frase preposicional é um bom exemplo de abstração. Se você alterar posteriormente a frase nominal para incluir os nomes das pessoas, a frase preposicional ainda funcionará.
    3. deve informar apenas um verbo como "pula" ou junte um verbo e um advérbio para criar combinações como "pula tristemente", "cochila rapidamente" ou "pensa sonolento".
    5. Estas frases estarão gramaticalmente corretas, mas não necessariamente farão sentido. Por exemplo, frase nominal pode gerar "uma pizza cansada".
  5. Em seguida, crie uma frase complicada de repórter que combine uma locução nominal, uma locução verbal e uma locução preposicional.
  1. Ocasionalmente, inclua nomes de pessoas em vez de uma frase nominal. Portanto, em vez de algo como "meu elefante bobo", o programa às vezes pode usar "Jamie" ou outros nomes.
  2. Frases nominais nem sempre precisam ter um adjetivo. Eles podem ter nenhum, ou dois, ou mais. Adicione alguma variedade.
