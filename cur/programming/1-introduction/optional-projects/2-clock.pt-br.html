Construa seu próprio relógio

Neste projeto, você modelará um relógio que exibe a hora atual. Seu código fará com que os ponteiros das horas, minutos e segundos girem corretamente e controlem o tempo.
  1. Para construir um relógio, você precisará calcular quantos graus o ponteiro dos segundos deve girar a cada segundo. Como o ponteiro dos minutos deve girar a cada minuto? De uma hora para a outra, quantos graus o ponteiro das horas deve girar?

  2. Este arquivo inicial oferece três sprites com fantasias para os ponteiros de segundos, minutos e horas. O ponteiro vermelho dos segundos também contém um arquivo de som para o "tique-taque do relógio".
    3. Você pode achar esses blocos úteis ao escrever o código para o ponteiro dos segundos.

  3. Tente, por conta própria, escrever o código para cada sprite, dizendo para onde apontar no momento atual. ConfiraCrie seu próprio relógio Dicas apenas se estiver realmente travado.
  1. Na maioria dos relógios com ponteiros, o ponteiro das horas se move continuamente em vez de pular uma vez a cada nova hora. Por exemplo, o ponteiro das horas aponta entre 12 e 1 quando a hora é 12:26:36. Se o ponteiro das horas pular de uma hora para outra, ajuste seu código para implementar esse movimento mais realista.

  2. Escolha ou desenhe figurinos para os ponteiros e o relógio para construir um relógio/relógio de designer em seu próprio estilo. Você pode desenhar imagens ou encontrá-las na web e usar um software de edição de imagens como o Adobe Photoshop para editá-las e orientá-las corretamente. Você precisará ancorar as fantasias corretamente colocando o centro de rotação no ponto apropriado.
  3. Crie um despertador que receba alguma entrada do usuário e emita um bipe ou toque quando o tempo acabar. Para armazenar a entrada do usuário, você precisará aprender como criar uma variável usando o menu Variáveis.
  4. Construa um “relógio de vovô” com um pêndulo oscilante em seu mecanismo. Ou um “Relógio Cuco” que tem um pássaro animado que aparece em intervalos regulares.
