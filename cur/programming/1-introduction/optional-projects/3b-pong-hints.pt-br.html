Dicas para o jogo de Pong


  1. Examine o arquivo inicial onde um sprite de remo retangular e um sprite de bola circular foram criados para você.< /li>
  2. Escreva o código para o remo primeiro. A raquete deve se mover apenas para cima e para baixo perto da borda direita da tela. Quando o usuário pressiona as teclas de seta para cima/para baixo no teclado, a raquete deve se mover para cima ou para baixo um certo valor, mas nunca deve sair das bordas da tela. Você pode achar os seguintes blocos úteis.
  3. O código da bola é um pouco mais complicado devido ao salto da raquete. Inicialize a bola para aparecer no meio da tela e dê a ela uma direção aleatória. Em seguida, verifique se a bola está na borda da tela ou tocando na raquete. Em cada caso, a bola precisa quicar. Snap! já fornece um bloco para ricochetear nas bordas da tela.

    Você pode escrever o código para a bola quicar na raquete percebendo que a bola quica na raquete com o negativo da direção em que ela atinge a raquete. Lembre-se que no Snap! os ângulos positivos são medidos no sentido horário a partir de 0°, direção para cima, e os ângulos negativos são medidos no sentido anti-horário a partir de 0°, direção para cima.
