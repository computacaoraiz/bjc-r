Fazendo um novo bloco

  1. Crie um novo bloco clicando em perto do final da paleta, clicando no sinal + no canto superior direito da paleta (mostrado abaixo) ou clicando com o botão direito do mouse (ou clicando com a tecla Control pressionada em um Mac) em um local vazio na área de script e escolha "criar um bloco..." (mostrado abaixo).
  2. Escolha a cor, neste caso vermelho para Listas.
  3. Digite o título para o seu bloco: fofoca reação.
  4. Selecione a forma e clique em OK. Para este bloco, escolha a forma oval reporter porque você deseja que ela informe um valor (a resposta escolhida) para outro bloco.

Como você escolheu a forma do repórter, o editor de blocos abre com um bloco já anexado ao bloco de chapéu. Qualquer entrada que você fornecer ao report se tornará o valor relatado pelo seu novo bloco.

Codificando o script do bloco

  1. Arraste os blocos que você precisa. Para seu novo bloco, use e . Use o valor relatado por list como uma entrada para item.
  2. Junte os blocos. A estrutura do código será exatamente como o código dentro de quem, faz o quê e quem 2.
  3. Preencha todas as entradas.
    • Para este projeto, clique na seta para baixo em item para alterar "1" para "aleatório".
    • Em seguida, preencha a lista com frases como "Adivinhe", "Eu ouvi" ou "Quem se importa!"
  4. Clique em OK quando estiver satisfeito.
: Procedimentos, Relatores e Comandos
AAP-3.A.1, AAP-3.A.2

Um procedimento é uma sequência nomeada de instruções que pode receber entradas e pode relatar um valor. Algumas linguagens chamam procedimentos métodos ou funções. Aqui estão dois tipos de procedimentos que você viu no Snap!:

  • Repórteres têm uma forma oval. Eles informam um valor.
  • Comandos têm a forma de um quebra-cabeça. Eles dizem ao computador para fazer algo sem informar um valor.

    AAP-3.A.7, AAP-3.C.2
  • O procedimento seria escrito como 
    PROCEDIMENTO duplo(x)
                        {
                            RETURN(2 * x)
                        }
    ou .

    Muitos idiomas (e o AP CS Principles Exam) usam return em vez de report como o nome do comando para retornar um valor no final de uma chamada de função.

    • AAP-3.A.9
  • O exame usa "valor de um procedimento" para significar o valor que o procedimento retorna quando chamado. Por exemplo, "o valor de 
    duplo(5)
    " significa 10.
