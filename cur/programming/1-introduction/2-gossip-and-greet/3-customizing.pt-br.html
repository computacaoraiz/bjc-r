Por que esses blocos usam ? Por que não apenas usar ?

Os blocos report e say significam coisas muito diferentes. Report informa algo ao computador, e esse valor pode ser usado como entrada para outro bloco, como o bloco join dentro de gossip. Say apenas mostra um valor para o usuário em um balão de fala.

: Listas, Strings, e Concatenação
    AAP-1.C.1 primeira frase
  • O que significa "ordenado"?
    Isso não significa que os itens devem aparecer em ordem alfabética ou ordem numérica. Significa apenas que é uma lista diferente de .
    A lista é uma sequência ordenada de itens. Você já viu este exemplo:
    • AAP-2.D.1, AAP-2.D.2
  • Os itens desta lista são strings. Uma string é uma sequência de caracteres (letras, dígitos, pontuação, etc.). Uma substring é apenas uma parte de uma string existente. Por exemplo, "Hanna", "anna" e "nnah" são substrings da string "Hannah". (A string vazia, assim como a string original, também são substrings.)
  • concatenar strings significa fazer uma string maior conectando duas ou mais strings menores. No Snap!, o O bloco permite concatenar strings.
