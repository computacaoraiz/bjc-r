Removendo Duplicatas

Nesta página, você aprenderá uma técnica recursiva para remover duplicatas de uma lista.

Suponha que você tenha uma lista de itens e deseje saber se os elementos da lista são distintos. Por exemplo, você pode querer ter certeza de que nada na sua lista de compras se repete.

Como primeiro passo, vamos apenas responder à pergunta sim/não: existem duplicatas na sua lista?

  1. Experimente com o bloco usando algumas listas de entrada diferentes. O que esse bloco relata?
  2. Termine de construir o predicado os itens são distintos?, que está iniciado abaixo. Como você está escrevendo um predicado, seu procedimento deve sempre relatar verdadeiro ou falso.
    Tente fazer isso sem esta dica.
    Se o programa chegar ao terceiro bloco report no final, o que isso lhe diz sobre os itens na lista?
    Observe que este procedimento chama a si mesmo no final. (É recursivo. Em vez disso, você deve usar um valor de entrada menor.
  3. Se você duplicasse o tamanho da lista, esse algoritmo levaria o mesmo tempo? O dobro do tempo? Mais do que o dobro do tempo?

Para sua lista de compras, você não gostaria apenas de saber se há duplicatas. Você gostaria de ter uma nova lista com as duplicatas removidas.

  1. Experimente com o bloco usando algumas entradas diferentes. O que esse bloco relata?
  2. Construa um relatório itens distintos de usando uma estrutura semelhante à do predicado os itens são distintos?.

    O algoritmo para este bloco tomará as mesmas decisões que no os itens são distintos?. Mas esse era um predicado. Este deve relatar uma lista. Portanto, examine os três blocos report em seu código para o predicado os itens são distintos? e decida o que eles devem relatar para o relatório itens distintos de.

    Precisa de outra dica?

    • Se a lista estiver vazia, o que o itens distintos de deve relatar?
    • Se o primeiro item da lista aparecer no resto da lista, não importa qual cópia você deixa de fora. Existe uma maneira fácil de obter uma versão da lista sem uma dessas cópias?
      • E se houver outras duplicatas nessa lista?
    • Se o computador chegar ao terceiro bloco report, o que isso lhe diz sobre o primeiro item da lista? Você deseja que o primeiro item faça parte da lista que você relata?

  3. Teste-o. Certifique-se de escolher bons casos de teste: mais de um par de duplicatas, mais de dois itens do mesmo valor, duplicatas lado a lado na lista, etc.
