Nesta página, você usará um algoritmo de busca binária para realizar uma busca eficiente em uma lista ordenada.

Responder a esta pergunta é simples no Snap!, porque você pode simplesmente consultar um dicionário: . No entanto, "simples" não significa rápido. O bloco contém ainda precisa verificar cada item na lista até encontrar o que você procura (e dizer verdadeiro) ou alcançar o final da lista (e dizer falso). Ainda é uma busca linear.

Quando você está procurando apenas uma coisa em uma lista (por exemplo, verificando se uma palavra está escrita corretamente) em vez de encontrar todas as palavras com alguma característica (por exemplo, procurando todas as palavras de cinco letras), você pode usar a estratégia da Página 1: Adivinhe o Número para tornar seu algoritmo mais rápido. A melhor estratégia para esse problema é um algoritmo de busca binária: sempre adivinhe o valor do meio e ajuste sua suposição com base em se ela foi alta ou baixa demais. Dessa forma, você elimina metade das possibilidades a cada suposição. Podemos usar uma estratégia semelhante para procurar uma palavra em uma lista de palavras.

Você poderia ter escrito um programa de adivinhação de números mais simples: o computador poderia adivinhar o número 1, depois 2, depois 3 e assim por diante até encontrar o número. Isso seria um algoritmo de busca linear; é mais fácil codificar, mas leva mais tempo para ser executado, porque a cada vez que faz uma suposição errada, elimina apenas essa suposição. Com a busca binária, cada suposição errada elimina metade das possibilidades de uma vez.
: Busca Binária
AAP-2.P.1, AAP-2.P.2

Um algoritmo de busca binária começa no meio de uma lista ordenada e elimina repetidamente metade da lista até encontrar o valor desejado ou até que todos os elementos tenham sido eliminados.

AAP-2.O.1
Você aprendeu sobre a travessia de uma lista em Unidade 2 Laboratório 2 Página 3: Verificando Cada Resposta de Quiz.

A busca linear faz uma travessia completa da lista. A busca binária economiza tempo fazendo uma travessia parcial da lista.

A única coisa que quase todos sabem sobre computadores é que eles usam o sistema binário: uns e zeros. A busca binária não tem nada a ver com isso. A palavra "binário" apenas significa "dois", quer se trate de dois dígitos ou duas metades.

    Para usar uma busca binária, os dados devem estar...
    binários
    Todos os dados em um computador são representados usando binários (uns e zeros), mas isso não tem nada a ver com buscas binárias, que comparam com o valor do meio para escolher qual metade eliminar.
    ordenados
    Correto! Se os dados estiverem ordenados, comparar com o valor do meio fornecerá boas informações sobre qual metade dos dados manter.
    desordenados
    Se os dados estiverem desordenados, você não pode ter certeza de que tudo antes ou depois do valor do meio pode ser eliminado.
    lineares
    "Linear" é o nome de outro algoritmo de busca, não uma propriedade dos dados.
    Quais das seguintes perguntas podem ser respondidas com uma busca binária, assumindo que os dados estão ordenados? Marque todas as que se aplicam:

    Temos que encontrar todas as músicas da Beyoncé, não apenas uma.
    Diga-me se o pão está na minha lista de compras.
    Correto! Você está procurando um item na lista.
    Quem na minha lista de contatos mora na Avenida Grand?
    Sua lista de contatos provavelmente está ordenada por nome, não por endereço. Além disso, pode haver mais de uma pessoa que mora na Avenida Grand.
